# 周末去哪儿 — 产品概述

## 一句话

像翻一本私人的旅行美食杂志，收录周末想去的地方。

## 解决什么问题

- 朋友推荐了好吃的店，微信聊天记录一翻就找不到
- 小红书/抖音刷到好玩的地方，回头就忘
- 周末不知道去哪儿，翻看之前的收藏找灵感
- 去过的和想去的混在一起，没有区分

## 核心场景

| 场景 | 操作 |
|------|------|
| 朋友分享了一个链接 | 粘贴 → 自动解析 → 打标签 → 加入收藏 |
| 路过一家店想记下来 | 拍照 → 选分类 → 填写名称 → 加入收藏 |
| 周末不知道吃什么 | 打开首页 → 每日精选随机推荐 → 找灵感 |
| 想吃某道菜 | 列表页搜索"水煮鱼" → 所有带这个标签的店都出来 |
| 出去探店 | 地图页按分类/标签筛选 → 看附近有什么 |
| 整理收藏 | 分类管理 → 自定义分类 → 编辑 |

## 功能清单

- **添加收藏** — 链接解析、相册/拍照上传封面、分类、标签、评分、位置、备注
- **首页画报** — 每日精选（随机推荐）+ 往期收藏（双列瀑布流）
- **地图浏览** — 水滴形定位针、按分类+标签筛选、名称常显
- **列表搜索** — 分类筛选、标签/名称模糊搜索、长按操作菜单
- **收藏详情** — 大图封面、位置导航、编辑、分享、复制备注
- **分类管理** — 自定义分类名和颜色、点击跳转筛选
- **标签统计** — 全部标签排行榜、点击即搜
- **启动页** — 七日刊风格，每天不同的杂志封面
- **编辑室设计** — 思源宋体 + 思源黑体、暖纸墨色、杂志排版

## 页面结构

```
启动页 → 首页（画报）
       → 地图页
       → 添加页
       → 列表页 → 详情页
       → 我的页 → 分类管理
                → 标签统计
```
