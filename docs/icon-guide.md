# Tab 图标资源准备指南

## 图标清单

所有图标尺寸：40x40px PNG（建议@3x = 120x120px 实际导出）
配色：默认态 #8B7355，选中态 #FF9A56

| 文件名 | 用途 | 建议图标 |
|--------|------|---------|
| `tab-home.png` | 首页 默认 | 房子 |
| `tab-home-active.png` | 首页 选中 | 房子（实心） |
| `tab-map.png` | 地图 默认 | 地图标记 |
| `tab-map-active.png` | 地图 选中 | 地图标记（实心） |
| `tab-add.png` | 添加 默认 | 加号圆圈 |
| `tab-add-active.png` | 添加 选中 | 加号圆圈（实心，橙色） |
| `tab-list.png` | 列表 默认 | 列表/菜单 |
| `tab-list-active.png` | 列表 选中 | 列表/菜单（实心） |
| `tab-mine.png` | 我的 默认 | 用户头像 |
| `tab-mine-active.png` | 我的 选中 | 用户头像（实心） |

## 其他图标

| 文件名 | 尺寸 | 用途 |
|--------|------|------|
| `default-cover.png` | 750x500px | 默认封面图（美食相关） |
| `marker-food.png` | 36x36px | 地图标记点图标 |

## 获取方式

1. **iconfont.cn** — 搜索对应图标，下载PNG格式
2. **Figma** — 使用社区图标库导出
3. **临时方案** — 使用纯色方块占位，后期替换

图标放置路径：`miniprogram/images/`
