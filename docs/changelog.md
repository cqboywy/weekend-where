# 周末去哪儿 — 更新日志

## v1.1 — 编辑室风格重设计

**整体 UI 重设计**
- 配色从暖桃色系改为暖纸+墨色+铁锈红
- 字体从 Playfair Display 改为思源宋体 Noto Serif SC（标题）
- 正文从 Source Sans 3 改为思源黑体 Noto Sans SC
- 圆角大幅减小，卡片更方正
- 阴影大幅减弱，靠留白和线条区分层级
- 次要按钮改为细线框透明底

**启动页**
- 从四季主题改为七日刊（每天不同杂志封面）
- 极简几何装饰标记（细线、圆环、点）
- 每天不同的配色和 slogan

**首页**
- 移除快捷入口和收藏统计，改为精选画报
- 每日精选大卡片（随机推荐）
- 往期收藏双列瀑布流
- 诗意时段短语替换"早上好/下午好"
- 卡片微叠层阴影（纸张切边感）

**图标系统**
- Lucide 图标描边从 2px 减为 1.5px
- Tab 栏图标重新生成（细线勾勒风格）
- App 图标（W 字母编辑室风格）

**页面润色**
- 添加页标题改为"· 新收藏 ·"
- 列表页标题精简
- 详情页按钮统一描边
- 我的页标签统计移至菜单上方
- 我的页加 WEEKEND·WHERE 刊头

## v1.0 — 核心功能

**收藏管理**
- 添加收藏（链接解析、手动录入）
- 编辑/删除收藏
- 标签系统（最多 10 个）
- 评分、备注、位置

**分类管理**
- 自定义分类名和颜色
- 16 个默认分类
- 点击跳转列表筛选

**地图浏览**
- 水滴形定位针
- 按分类和标签筛选
- 名称常显

**列表搜索**
- 分类筛选
- 名称/标签模糊搜索
- 长按操作菜单

**卡片展示**
- 首页双列瀑布流
- 收藏卡片显示前 3 个标签
- 无图时自动生成分类光斑封面

**封面图片**
- 相册/拍照上传
- 链接解析自动获取 OG 图片
- CloudBase 云存储

**标签统计**
- 列表页热门标签（已移除）
- 我的页全部标签排行榜
- 点击跳转搜索

**体验优化**
- iPhone 灵动岛适配
- 底部 home indicator 避让
- 编辑收藏表单清空逻辑修复
- 上传状态残留修复
